The ``my_list`` module
=====================
Using ``my_list``
----------------------
First import ``my_list`` from the ``1-my_list`` file:

    >>> MyList = __import__('1-my_list').MyList

    Test: positive numbers
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 5]

    Test: negative numbers
    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(-3)
    >>> my_list.append(-1)
    >>> my_list.append(-2)
    >>> print(my_list)
    [-5, -3, -1, -2]

    Test: sort out my MyList
    >>> my_list.print_sorted()
    [-5, -3, -2, -1]

    Test: Empty list
    >>> my_list = MyList()
    >>> print(my_list)
    []
